# House Price Predictor from Scratch

This project implements a house price prediction model from scratch using Python and NumPy.

The goal is to understand the core concepts of Machine Learning by building the model step by step, without relying on high-level libraries.

---

## What this project covers

This notebook revisits and connects the following concepts:

- Supervised Learning
- Linear Regression
- Prediction Function
- Cost Function
- Gradient Descent
- Multiple Features
- Vectorization
- Feature Scaling
- Polynomial Regression

---

## Key idea

Instead of using pre-built machine learning libraries, this project focuses on understanding how the model works internally:

- how predictions are computed
- how error is measured
- how parameters are optimized
- why feature scaling improves convergence
- how polynomial features increase model flexibility

---

## Visualizations

The notebook includes:

- cost function over iterations
- comparison between scaled and non-scaled training
- cost function contours (ellipses)
- gradient descent trajectory on the cost surface

---
